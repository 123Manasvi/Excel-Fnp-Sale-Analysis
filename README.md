# Excel-Fnp-Sale-Analysis
Overview

This project presents a Sales Analysis Dashboard for Ferns N Petals (FNP) built using Microsoft Excel.
The dashboard analyzes sales data to uncover insights about revenue trends, customer behavior, product performance, and delivery efficiency.

The goal of this project is to demonstrate data analysis, data modeling, and dashboard visualization skills using Excel.

Dashboard Insights

The dashboard highlights several key business metrics:

Total Orders: 1000

Total Revenue: ₹35,20,984

Average Order to Delivery Time: 5.53 days

Average Customer Spending: ₹3,520.98

It also provides visual insights for:

Revenue by Occasion

Revenue by Product Category

Revenue by Order Time (Hourly Trends)

Revenue by Month

Top 5 Products by Revenue

Top 10 Cities by Number of Orders

Interactive filters allow users to analyze data based on:

Delivery Date

Order Date

Occasion
